# 双轨用法 + 映射表接口

> 双轨 = 外部工具（AI）分析内容 + skill 脚本执行格式化。

---

## 工作流

```
外部 AI（语义分析）→ 产出映射表（.txt）→ inject_marks.py（写入 [N]）→ refcite.py（[N] → REF 域）
```

AI 只做语义判断（哪段引哪篇），脚本做文本操作。各司其职。

---

## 使用建议

| 方面 | 说明 |
|------|------|
| **映射表质量** | 映射越准确，最终引用越可靠。建议通读确认 |
| **文献编号** | 映射表编号需与文末参考文献列表一一对应 |
| **句级精度** | 建议精读目标段落后再决定插入位置 |
| **首次使用** | 先测试 2-3 段，确认效果后再处理全文 |
| **保存后校验** | `refcite.py --replace --verify` 自动检查书签和 REF 域结构 |

## 索引规则

段落索引 = `doc.paragraphs` 数组中的 0-based 位置，**包含所有段落**（空行、标题、封面等）。

```python
doc.paragraphs[0]  → 文档第 1 段
doc.paragraphs[19] → 文档第 20 段
```

过滤无效段落是脚本内部的事，映射表只需给出原始索引。

## 映射表格式（给 inject_marks.py 用）

### 句级（推荐）

```
P7:3:。[5]     段落7 第3句 句号前 插 [5]
P14:6:。[1]    段落14 第6句 句号前 插 [1]
P23:2:；[7]    段落23 第2句 分号前 插 [7]
```

每行格式：`P{段落}:{第几句}:{标点}[{文献号}]`

- `P{段落}` — 0-based 段落索引
- `{第几句}` — 1-based，按标点拆分
- `{标点}` — 通常用 `。`，支持 `。！？；，`
- `[N]` — 与文末参考文献列表编号一致

### 段级

```
P7:[5]     段落7 段尾 插 [5]
```

需加 `--para` 参数。

## 完整操作流程

```bash
# 0. 清空旧标记（换方案时必做）
python scripts/inject_marks.py 论文.docx --strip

# 1. 注入新标记
python scripts/inject_marks.py 论文.docx --map-file plan.txt

# 2. 替换为 REF 域
python scripts/refcite.py 论文.docx --replace --verify
```

### 当正文已有 [N] 标记时

如果文档已有手打 `[N]` 标记（如 [1]~[6]），需补充更多引用：

1. AI 分析正文 → 产出 `plan.txt`
2. 运行：
   ```bash
   python scripts/inject_marks.py 论文.docx --strip
   python scripts/inject_marks.py 论文.docx --map-file plan.txt
   python scripts/refcite.py 论文.docx --replace --keep-numbers --verify
   ```

### 编号策略

| 参数 | 效果 |
|------|------|
| 不加 `--keep-numbers` | 按首次出现顺序重排编号 |
| 加 `--keep-numbers` | 保持原编号不变 |

## 外部 AI 对接指南

### 文档解析

```python
import docx
doc = docx.Document('论文.docx')
for i, p in enumerate(doc.paragraphs):
    print(f'[{i}] {p.text}')
```

### 段落分析

逐段判断：
1. 段落是否在讨论某篇文献的研究对象或观点？
2. 段落是否在比较多篇文献？
3. 段落是否引用某篇文献的方法或结论？

### 信息获取方式

对不熟悉的文献（新期刊、非公开方法、不确定结论），主动询问用户要不要搜索确认。不搜则凭训练知识判断。

搜索不是双轨的必要条件。AI 自行决定每条文献是否需要搜索，不让用户做全局决策。

### 句级分析 → 产出映射表

按句子分析，**逐句判断引用归属**。即使整段只引一篇文献，也定位到最相关的那句话的句末，不是段末。

跨文献对比段落（一段话讨论多篇文献），按句号/分号拆到句子级别：

```
段落 P14 两句：
  句1: "李长莉在《晚清上海社会的变迁》中将…转型；" → 文献 #2
  句2: "王笛以近代成都茶馆为个体研究对象…互动。" → 文献 #1
```

产出映射表：
```
P14:2:；[2]
P14:3:。[1]
```

### 常见注意事项

| 场景 | 建议 |
|------|------|
| 摘要段落 | 通常不含引用，排除 |
| 封面/标题行 | 排除 |
| 参考文献列表本身 | 排除 |
| 对文献不熟悉 | 搜索或用户提供信息补充 |
| 多篇文献在同一段 | 拆为句级映射 |
| 不确定映射是否准确 | 仅映射确定的部分，留空不映射 |
| **作者名 ≠ 需要引用** | 段落中出现作者名不代表内容出自该文献 |
| **传记信息不引** | 出生日期、职位等公开信息不属可引用内容 |
