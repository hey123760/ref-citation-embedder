# 参考文档

> 编号体系、安全边界、Run 拆分原理等技术细节。

---

## 项目编号体系

严格区分「项目编号」和「简单编号」：

| 概念 | 含义 | 后果 |
|------|------|------|
| **简单编号** | 固定编号，1号永远是第一篇文献 | 文献列表顺序与引用顺序无关 |
| **项目编号** | 编号 = 正文首次出现顺序 | 删除文献会级联顺延 |

### 三条铁律

**铁律一：编号 = 首次出现顺序**

```
正文第1段引用的文献 → [1]
正文第2段引用的文献 → [2]
...文献的原始编号在匹配后抛弃，只保留内容
```

**铁律二：文献列表按编号重排**

参考文献列表的输出顺序 = 正文引用顺序，不是用户提供的顺序。

**铁律三：增删文献必须全量重跑**

```
旧: [1] [2] [3] [4] [5]    删除第3条
新: [1] [2] [4] [5]        ❌ 不连续
新: [1] [2] [3] [4]        ✅ 全量重排
```

### 特殊情况

- **同一篇多段引用**：编号不变，不重复编号
- **文献总数 > 段落数**：未匹配文献追加到末尾，注明未引用
- **手动删除中间一条**：必须全量重跑，编号顺延

## 安全边界

### 操作范围

| ✅ 可以修改 | ❌ 绝不修改 |
|-----------|-----------|
| 参考文献段落（重排） | 脚注/尾注的任何内容 |
| 正文匹配段落（追加 REF 域） | 页眉/页脚/分节符 |
| `numbering.xml`（仅动态分配空闲 ID） | 样式表（styles.xml） |
| 书签（bookmarkStart/End） | 文档主题/字体/颜色 |
| REF 域（fldChar/instrText） | 原文文本内容 |

### 自动编号安全

- 书签 ID 从 5000 开始分配，避开文档既有书签
- numId 从 10 开始扫描，找第一个未被占用的 ID
- 绝不使用 `numId=0`、`numId=1` 等系统低位 ID
- 自动编号与脚注号互不冲突（不同 numId）

### 重跑安全

- 每次运行前清理已有 REF 域
- 删除旧参考文献列表，全量重建
- 不清除其他任何元素

## Run 拆分机制

### 为什么需要拆分

Word 文档中文本存储在 `<w:r>`（run）元素中。一个 run 可能包含多个字符和标点。REF 需要插在标点之前，就必须在一个 run 中间切开。

### 拆分流程

```
原始 run:  text： ，，
                   ↑
              目标标点
                   ↓
拆分后 run A:  text：      ← 标点前文本（含标点）
拆分后 run B:   ，，       ← 从标点开始（含标点）
               ↑
           在此插入 REF
```

### 特殊情形

- **标点在 run 末尾**（如 `认为，` 独立成 run）：无需拆分，直接在原 run 后插入 REF
- **标点不在 `w:t` 中**（如 `<w:sym>` 特殊符号）：跳过该 run，继续找下一个含标点的 run
- **段末标点**：段落以 `，。` 结尾时，REF 插在标点前。段落结尾非标点时，REF 插在最后一个 run 后

### 吸附规则

所有匹配模式统一吸附到后面首个 `，` 或 `。` 之前，搜索范围限 30 字符内。
书名号、问号、叹号等不作为插入点考量。

## 脚注检测

段落级检测 `<w:footnoteReference/>`，含脚注段落跳过引用匹配。

**已知局限**：当前为段落级检测。含脚注的段落即使有多句正文也会被整体跳过。
更精确的句级检测（仅跳过含脚注的句子）未实现。

---

## 备份去重

- 固定文件名 `{原名}.bak.docx`，无时间戳
- 每次运行检查同名备份是否存在且大小一致，存在则跳过
- 仅首次运行创建备份
- 如需保留某版本备份，请手动重命名

## 格式规范

### 参考文献格式（GB/T 7714-2015）

- 期刊论文：作者. 题名[J]. 刊名, 年, 卷(期): 起止页码.
- 学位论文：作者. 题名[D]. 城市: 学校, 年.
- 英文论文：Author. Title[J]. Journal, Year, Volume(Issue): Pages.

### 字体字号

- 正文引用标记：宋体 + Times New Roman，12pt（sz=24），上标，黑色
- 参考文献正文：宋体 + Times New Roman，12pt
- 无超链接蓝色/下划线样式
